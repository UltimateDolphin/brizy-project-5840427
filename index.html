You can also access handshake domains through the Private Internet Access VPN.

Hns.to: The Easy Access Solution

Hns.to is a handshake website viewer that requires no downloads and no browser extensions. Simply add hns.to to the front of any handshake website in the example format: hns.to/game.installnow or add .hns.to to the end of any handshake website such as app.installnow.hns.to. Adding hns.to to the front is recommended as this action is anchored in adding http:// to the front of websites.

Important Notes

Two important points to note when using handshake domains.

Don't Collect Sensitive Information

SSL is currently not supported by handshake so be mindful of what data you collect via the site if any. Directing users to 3rd party systems to make contact or to process data is recommended. Examples: Click to email, Discord, Telegram, Send a Facebook Message.

Email Is Available Through Hmail

Visit Hmail.app to setup private email on any handshake domain. The current cost is 2.5 hns per month for private email using any handshake domain name you own. Examples john@stationary.shopnow, mary@flowers.shopnow
